# Nigerian-Food-Identification-project-with-Pytorch
The model was built with Pytorch ResNet9 architecture only with minor differences.

The model can name any image belonging to
- Jollof Rice
- Fried Rice
- Moimoi
- Pounded Yam
- Akara balls
